記一記帳是一款可以幫助使用者節省的記帳 App，透過每月的目標和花費的記錄，讓使用者可以清楚的管理自己的財務狀況，從而省下不必要的花費。

記一記帳還提供使用者掃描發票的 QR-code 和使用 Siri 來記帳，讓記帳更簡單快速。
